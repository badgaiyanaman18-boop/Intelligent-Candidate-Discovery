# Intelligent Candidate Discovery using AI

## Overview
This project ranks candidates against a Job Description using AI-powered semantic matching.

## Features
- Job Description Understanding
- Candidate Evaluation
- Semantic Similarity Matching
- Candidate Ranking
- Explainable Results

## Technologies
- Python
- Pandas
- Scikit-Learn
- Sentence Transformers
- Streamlit

## Workflow
Job Description → Embeddings → Similarity Scoring → Ranking → Explanation

## Output
Ranked list of candidates with relevance scores.
